#1 - Smart Swing Trade Execution Engine

What & Why

Replace the current signal-only scheduler with a full swing trade execution engine that opens real positions on Deriv with dynamic take profit, trailing stop, and a 3-layer exit system. Also expand supported instruments beyond just Boom/Crash to include Volatility and Jump indices.

Done looks like

When a signal fires, a position is opened on Deriv with lot size based on 20-25% of equity
Take profit is dynamically calculated at trade entry using signal confidence score × ATR × historical average move for that pattern
A trailing stop follows price as it moves in favour, locking in gains
Three exit conditions: TP hit, trailing stop triggered, or hard 5-day time limit
At 72 hours, if trade is profitable it closes; if in small loss it extends 24h at a time up to 5 days
Maximum 3 trades open simultaneously, 75-80% of equity deployed across them
New instruments supported: Volatility 75, Volatility 100, Jump 75, Step Index, Range Break 200
All open positions visible in the dashboard with live P&L, current SL, TP, and time remaining
Out of scope

MT5 integration
Crypto arbitrage
Settings UI panel (separate task)
Tasks

Deriv execution layer — Add buy/sell/close position methods to the Deriv client using the WebSocket trade API, supporting lot sizing and SL/TP placement.
Dynamic TP calculator — Build a function that at signal time computes the TP distance using: signal confidence score (from the ML model), current ATR, and the historical average move seen in past similar signals from the backtest database.
Trailing stop manager — Implement a per-position trailing stop that updates the SL on Deriv as price moves in favour, locking in a configurable % of peak floating profit.
3-layer exit logic — Wire up the exit conditions: TP hit (Deriv handles), trailing stop triggered, and time-based exit (72h check with 24h extensions up to 5 days).
Position sizing engine — Calculate lot size per trade as 20-25% of account equity, capped so no more than 3 simultaneous trades and 80% total equity deployed.
Expand instrument support — Add Volatility 75, Volatility 100, Jump 75, Step Index, and Range Break 200 to the symbol list, backfill candle history, and wire them into the scheduler and strategies.
Dashboard positions panel — Add a live positions panel to the frontend showing each open trade: instrument, entry price, current price, floating P&L, current SL, TP, and hours remaining before time exit.

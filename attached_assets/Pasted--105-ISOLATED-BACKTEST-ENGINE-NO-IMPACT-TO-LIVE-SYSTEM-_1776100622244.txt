#105 — ISOLATED BACKTEST ENGINE (NO IMPACT TO LIVE SYSTEM)

## Objective

Build a completely isolated backtesting system that:

- Uses existing V3 engines EXACTLY as they are (no changes)
- Runs deterministic historical simulations
- Implements the hybrid exit model (leg1 + momentum + regression)
- Outputs full trade logs and performance summaries
- Does NOT modify any live trading logic, scoring, or engine behaviour

This is a read-only simulation layer.

# PART 2 — HYBRID TRADE MODEL

## T004 — Implement hybrid exit system

Trade lifecycle:

### Phase 1 — Leg 1 target

- Define baseline move (use engine expected move or fallback fixed %)
- Example: 25–40% of expected move

When reached:
- evaluate continuation

---

### Phase 2 — Momentum extension

If momentum continues:
- allow trade to run further

If not:
- close at leg 1

---

### Exit conditions (first wins)

1. Leg 1 hit + weak continuation → exit
2. Strong continuation → extend trade
3. Aggressive regression:
   - price reverses beyond threshold
4. Hard stop loss:
   - fixed % OR volatility proxy
